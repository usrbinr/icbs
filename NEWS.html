  • Initial development version
  • Added theme_ibcs() for IBCS-compliant ggplot2 theme
  • Added ibcs_colors() for standardized color palette
  • Added scenario notation scales: scale_fill_ibcs_scenario(), scale_color_ibcs_scenario()
  • Added variance visualization: ibcs_variance_chart(), scale_fill_ibcs_variance()
  • Added ibcs_waterfall() for waterfall charts
  • Added ibcs_bar_chart() for bar charts with scenario notation
  • Added multi-panel layout functions: one_to_split_to_two_layout(), two_to_merge_to_one_layout()
  • Added make_connector() for standalone connector elements
  • Added add_panel_labels() for patchwork plot labeling
  • Added marquee integration: marquee_color_labels(), geom_marquee_colored()
